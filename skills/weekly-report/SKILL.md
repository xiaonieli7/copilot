# 技能名称: 周报生成器 (Weekly Report Generator)

## 📝 描述

这个技能帮助你快速生成结构化的工作周报，清晰展示本周完成的工作、项目进度、遇到的问题以及下周计划。

## 🎯 适用场景

当用户提到以下关键词时，你应该使用此技能：
- "写周报"
- "生成周报"
- "weekly report"
- "本周总结"
- "工作汇报"
- "周进度报告"

- ...

## 📋 使用指南

### 如何高效生成周报

1. **提前准备**:
   - 每天记录完成的工作（使用 TODO 工具或笔记）
   - 收集关键数据和指标
   - 记录遇到的问题

2. **内容要点**:
   - 用数据说话（完成X个功能、提交X行代码）
   - 突出关键成果，而非流水账
   - 诚实反映问题，提出解决方案

3. **避免**:
   - 过于简单（"本周完成了开发工作"）
   - 过于啰嗦（逐行描述每个小改动）
   - 只说问题，不提解决方案
